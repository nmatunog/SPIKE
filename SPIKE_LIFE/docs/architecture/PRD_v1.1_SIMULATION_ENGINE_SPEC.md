1. Design Philosophy
SPIKE LIFE is not intended to simulate perfect reality.
It should simulate:
* Realistic financial behavior
* Realistic consequences
* Understandable tradeoffs
* Teachable moments
Every mechanic must reinforce one of five lessons:
1. Cash Flow Matters
2. Protection Matters
3. Goals Need Planning
4. Wealth Requires Time
5. Advisors Create Value

20. Educational Success Criteria
After completing SPIKE LIFE, participants should be able to:
Explain cash flow.
Identify assets and liabilities.
Recognize protection gaps.
Understand FNA principles.
Design basic financial recommendations.
Explain the value of financial advice.
Connect life decisions with financial outcomes.
